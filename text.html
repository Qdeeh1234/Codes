01

الهيكل الأساسي والطباعة

في لغة C++، نعتمد على مكتبة iostream للتعامل مع الإدخال والإخراج. أحد أهم الفروقات التي يجب فهمها هو الفرق بين طرق إنهاء السطر، حيث يؤثر ذلك على أداء البرنامج. 


#include <iostream>
using namespace std;

int main() {
    // الطريقة الأسرع
    cout << "Hello World\n"; 

    // الطريقة الأبطأ (مع flush)
    cout << "End of Line" << endl; 
    
    return 0;
}

مقارنة الأداء: \n vs endl

\n ينزل سطراً فقط (سريع جداً)
endl ينزل سطراً + يفرغ الذاكرة (أبطأ)

اختيار اسم المتغير وفهم حجم البيانات في الذاكرة أمران حاسمان. استخدم الأداة أدناه للتحقق من صحة أسماء المتغيرات، وراجع الرسم البياني لمعرفة أحجام أنواع البيانات المختلفة.

مدقق أسماء المتغيرات

النتيجة ستظهر هنا...
  • لا يمكن أن يبدأ برقم.
  • لا يحتوي على مسافات.
  • لا يحتوي على رموز خاصة (ماعدا _).
  • غير مسموح بالكلمات المحجوزة (مثل int).
